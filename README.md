# Romyk Project

[Live Demo](https://emantarek48.github.io/Romyk-project/)

---

## Description
Romyk Project is a modern React-based single-page website showcasing multiple sections such as Hero, About, Featured, Services, Blog, and Footer.  
The project uses React Router for smooth navigation between sections and Bootstrap for responsive styling.

---

## Features
- **Hero Section:** Eye-catching introduction to the website.
- **About Section:** Information about the project or company.
- **Featured Section:** Highlights key features or projects.
- **Services Section:** Describes available services.
- **Blog Section:** Latest updates and posts.
- **Footer:** Contact info and navigation links.
- **Responsive Design:** Works perfectly on mobile, tablet, and desktop.
- **Smooth Navigation:** Easy to navigate between sections via Navbar.

---

## Technologies Used
- **React** – Front-end framework
- **React Router DOM** – Navigation between sections
- **Bootstrap 5** – Responsive design and components
- **Bootstrap Icons** – Vector icons
- **CSS / HTML** – Styling and layout
